# Aishwarya A - Portfolio Website

A modern, terminal-inspired portfolio website showcasing Site Reliability Engineering expertise and projects.

## Features

- **Terminal-Inspired Design**: Dark theme with terminal aesthetics and command-line styling
- **Smooth Animations**: Typing effects, fade-ins, and parallax scrolling
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Elements**: Hover effects, smooth navigation, and dynamic particles
- **Performance Optimized**: Lazy loading and efficient animations

## Sections

1. **Hero**: Eye-catching introduction with typing animation
2. **About**: Professional summary, achievements, and technical skills
3. **Experience**: Timeline of professional roles and accomplishments
4. **Projects**: Showcase of key projects with technologies and metrics
5. **Contact**: Multiple ways to connect

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Intersection Observer API
- Smooth Scroll API

## How to Use

1. Open `index.html` in a modern web browser
2. Navigate using the top menu or scroll through sections
3. All links are functional and ready to be customized

## Customization

To personalize the portfolio:

1. Update contact information in `index.html`
2. Modify color scheme in `styles.css` (`:root` variables)
3. Add your own projects and experience details
4. Replace placeholder links with your actual profiles
